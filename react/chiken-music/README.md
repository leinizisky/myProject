## 搭建react项目

### 全局安装create-react-app
npm install create-react-app -g

### 创建项目
create-react-app  项目名

### 多个端口驱动多个项目
port=端口号 npm start

### 安装依赖
cnpm install react-router-dom --save
cnpm install less-loader less --save
